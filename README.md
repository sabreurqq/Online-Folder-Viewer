Online-Folder-Viewer
====================
此為檔案瀏覽器線上版，可用於瀏覽遠端電腦上的檔案系統。<br/>
目前僅測試於JBoss7 + Chrome + Windows， 其他環境不保證可正常運行。

### 功能特色
* 支援單/多筆上傳與拖曳上傳
* 支援下載
* 可搜尋當前資料夾
* 可排序當前資料夾
